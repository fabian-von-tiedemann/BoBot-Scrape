# RUTIN
Prioriteringslista vid personalbrist
## Inledning
Det kan ibland förekomma sådan personalbrist så att alla arbetsuppgifter
inte kan utföras som planerat. Vid sådana tillfällen är det viktigt att
personalen inom Serviceboende känner till hur de ska prioritera och därför
finns en prioriteringslista framtagen
## Roller och ansvar
Enhetschef ansvarar för att
➢ göra prioriteringslistan känd för medarbetarna samt övergripande
arbete.
Medarbetarna ansvarar för att
➢ följa den akuta prioriteringslistan.
## Område
Prioritering av arbetsuppgifter inom Serviceboende i samband med
personalbrist.
## Utförande
Prioritering
Om det inte går att få in en timvikarie behöver arbetsuppgifterna ses över.
Mat- och omvårdnadsinsatserna ska prioriteras. Serviceinsatser kan
bokas om vid behov. Det är viktigt att meddela brukaren att insatsen flyttas
eller ändras på.
Akut
1. Mat-och omvårdnadsinsatser ska prioriteras. Prioritera vilka insatser
som ska utföras och vilka insatser som kan planeras in till en annan
dag.
2. Dra ner tiden hos brukarna genom att exempelvis slå ihop insatser,
utföra insatser via telefonsamtal.
3. Säkerställa och kontrollera att alla brukare fått nödvändiga insatser
efter dagens slut.

4. Dela på timvikarier mellan enheterna.
5. Ge nya timvikarier nödvändig introduktion för att kunna utföra
insatserna.
6. Beordra in ordinarie personal.
7. Samverka med anhöriga.
8. Dokumentera ej utförda insatser.

Förebyggande
1. Säkerställa att medarbetarna känner till rutin för när
Bemanningsenheten respektive Trygghetsjouren ska kontaktas.
2. Säkerställa att enheten har tillgång till timvikarier att ringa in vid
personalfrånvaro
3. Stärka krisberedskapen genom att arbeta förebyggande med
egenkontroller
4. Samverka med anhöriga och frivilligorganisationer för avlastning
förebyggande

Övergripande
1. Slå ihop flera insatser i samma besök.
2. Förkorta introduktionen för nya medarbetare.
3. Utför de allra nödvändigaste insatserna.
4. Samverka med övriga enheter och hjälps åt över gränserna.
5. Samverka med anhöriga, frivilligorganisationer och övriga kontakter
som finns.
6. Se över hur vi ska få in mer personal. Eventuellt fördela personal
resurser inom hela Vof
Stöd i planering
1. Delegering: Se till att brukare tar sin medicin.
2. Bokade möten: Om det är bokade möten som personalen ska följa
med hyresgästen. Exempel: Läkare, LSS handläggare,
Försäkringskassan e.t.c.
3. Brukare som sällan tar emot stöd: Hyresgäster som nästan aldrig
tar emot stöd eller har bara en gång i veckan.
4. Matlagning: Hyresgäster som behöver stöd i matlagning.
5. Nya hyresgäster: Hyresgäster som är nya på boendet ska i största
mån inte prioriteras bort. De måste bygga upp trygghet och tillit till
personalen.
6. Handling: Hyresgäster som behöver stöd vid handling. Försök om
möjligt att flytta handlingen till senare på dagen eller en närliggande
