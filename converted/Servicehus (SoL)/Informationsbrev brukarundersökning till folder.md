# INFORMATIONSBREV

Brukarundersökning för äldre – Socialstyrelsen

Socialstyrelsen genomför årliga brukarundersökningar inom hemtjänsten för att få insikter om hur äldre upplever den vård och omsorg de får. Resultatet från undersökningarna används för att utveckla och förbättra äldreomsorgen. Dina och andra personers synpunkter är mycket viktiga. De hjälper oss att förstå vad som fungerar bra och vad vi behöver förbättra.

Hur arbetar vi med resultatet?

När resultaten kommer går vi igenom dem tillsammans i verksamheten och jämför med tidigare år. Utifrån det tar vi fram förbättringsåtgärder så att hemtjänsten blir ännu bättre för dig som brukare. Vi återkopplar resultat och åtgärder till dig genom brev, fikaträffar och vår hemsida botkyrka.se.

Tack på förhand för att du hjälper oss att bli bättre!
