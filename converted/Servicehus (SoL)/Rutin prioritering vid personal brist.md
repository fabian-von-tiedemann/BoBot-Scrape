# RUTIN
Prioriteringslista vid personalbrist inom
Hemtjänsten
## Inledning
Det kan ibland förekomma sådan personalbrist så att alla
arbetsuppgifter inte kan utföras som planerat. Vid sådana tillfällen är
det viktigt att personalen känner till hur de ska prioritera och därför
finns en prioriteringslista framtagen.
## Roller och ansvar
Enhetschefer ansvarar för att göra prioriteringslistan känd för
medarbetarna.
Enhetscheferna ansvarar för förebyggande samt övergripande
arbete.
Medarbetarna ansvarar för att följa den akuta prioriteringslistan.
## Område
Beskriv vad som ska göras.
## Utförande
Prioritering
Om det inte går att få in en timvikarie behöver kanske jobben delas
om.
- Det är omvårdnadsinsatserna som ska prioriteras.
- Serviceinsatser kan bokas om vid behov.
- Egentiden kan bokas om vid behov.
- Det är viktigt att meddela brukaren att insatsen flyttas eller
ändras på samt dokumentera detta i journalen.
- Den som har brukaren på sitt schema är ansvarig att meddela
brukaren.

Förebyggande

1. Säkerställa att enheten har gott om timvikarier att ringa in vid
personalfrånvaro
2. Stärka krisberedskapen genom att arbeta förebyggande med
egenkontroller

2/2
3. Samverka med anhöriga och frivilligorganisationer för
avlastning förebyggande
4. Flera trygghetskameror installeras

Akut

1. Gå över till helgbemanning, trygghetsringning istället för
tillsynsbesök.
2. Prioritera vilka insatser som ska utföras och vilka insatser
som kan planeras in till en annan dag.
Omvårdnadsinsatser prioriteras.
3. Säkerställa och kontrollera att alla brukare fått nödvändiga
insatser efter dagens slut.
4. Sköta brukare som har dubbelbemanning i sängen med
enkelbemanning.
5. Dela på timvikarier mellan hemtjänstområdena.
6. Förkorta introduktionen för nya timvikarier.
7. Beordra in ordinarie personal.
8. Samverka med anhöriga.

Övergripande
Om personalbristen består över en längre tid.

1. Förkorta introduktionen för nya medarbetare.
2. Gå ned till helgbemanning och utför det allra nödvändigaste
insatserna.
3. Samverka med övriga enheter och hjälps åt över gränserna.
4. Samverka med anhöriga, frivilligorganisationer och övriga
kontakter som finns.
5. Fördela personalresurser inom hela Vof.
