## Inledning
Botkyrka kommun erbjuder brukare med den beviljade insatsen tillsyn att få
möjlighet till nattillsyn via trygghetskamera. Skillnaden mot personlig
nattillsyn är att personalen gör tillsynen via en kamera istället för via ett
hembesök. Denna rutin är till för att tydliggöra vem som gör vad i dessa
ärenden.

## Roller och ansvar

- Biståndshandläggare tar beslut om insatsen tillsyn samt lämnar besked
om avslut
- EC / Koordinator tar emot beställning och verkställer samt avslutar
verkställighet.
- EC / Koordinator informerar larmgruppen
- Larmgruppen hanterar den praktiska installationen och avslutet
- EC / Koordinator schemalägger e-tillsyn i Phoniro.
- IT-funktion / Vof ansvarar för den tekniska miljön,
systemadministration,  intern support, avtal samt drift- och
leverantörsuppföljning.

## Område
Vad föregår installationen av trygghetskamera, hur gör man samt vad händer
vid avslut.

På Botweb finns:

- informationsbladet ”Trygghetskamera” som riktar sig till brukare.
- Samtyckesblankett som ska skrivas under av brukaren.
- bilaga 1 som tydliggör vilka tider som avses och som är underskriven
av EC.
- bilaga 2 som tydliggör vad nattillsyn är.
- bilaga 3 som informerar vart brukaren kan vända sig om problem
uppstår.
- Ansvarsförbindelse med tillhörande bild på vad paketet med kameran
innehåller. Denna ska skrivas under av brukaren.

## Utförande

Beslut, beställning och installation

1. Biståndshandläggare tar beslut om insatsen tillsyn och skickar beslutet i
verksamhetssystemet.

2. EC / Koordinator tar emot beställningen i verksamhetssystemet.

3. EC / Koordinator meddelar ni larmgruppen via verksamhetssystemet.

4. Larmgruppen kontaktar brukaren och kommer överens om tid för
installation samt tider för tillsyn.

5. Om det endast finns denna insats behöver nycklar inhämtas enligt
ordinarie nyckelrutin.

6. Larmgruppen skriver ned antal överenskomna tillsyner och tidsplan i
bilaga 1 (underskriven av EC) i två exemplar. Ett sparas hos
larmgruppen tillsammans med nyckelkvittens.

7. Vid installation av trygghetskamera får brukaren muntlig och skriftlig
information samt skriver under samtyckesblanketten och
ansvarsförbindelsen. Larmgruppen lämnar kvar bilaga 1 som är
underskriven av EC samt bild på vad paketet med kameran innehåller
och bilaga 3 som informerar om vart brukaren kan vända sig vid
funderingar.

8. Larmgruppen informerar EC/Koordinator at det är klart för start av
tillsyn.

9. EC/Koordinator verkställer beställningen.

10. EC / Koordinator schemalägger e-tillsyn i Phoniro.

11. Koordinator uppdaterar genomförandeplanen med tillsynstider (Kan
delegeras till nattpersonal).

Avslut och avinstallation

1. Biståndshandläggaren skickar meddelande till EC/Koordinator när
insatsen tillsyn ska avslutas.

- Om utföraren får informationen först måste de be brukaren kontakta
biståndshandläggaren så att det sker på ett korrekt sätt.

- Om brukaren inte har möjlighet att göra detta ringer koordinator
eller annan personal till myndighet med informationen.

- Biståndshandläggaren ansvarar för att säkerställa att insatsen tillsyn
ska avslutas och meddelar sedan utföraren i vanlig ordning.

2. EC / Koordinator avslutar e-tillsyn i Phoniro

3. EC / Koordinator meddelar larmgruppen via meddelandehanteringen i
verksamhetssystemet om att trygghetskameran ska avslutas hos
brukaren.

4. Larmgruppen kontaktar brukaren för att boka tid för avhämtning av
kamerapaketet.

5. Larmgruppen hämtar paketet med trygghetskameran hos brukaren och
meddelar EC/Koordinator att det är gjort.

6. EC/Koordinator avslutar ärendet i verksamhetssystemet.
