# RUTIN
Rutin för installation och avslut av
trygghetskamera inom Hemtjänst

## Inledning
Botkyrka kommun erbjuder brukare med den beviljade insatsen tillsyn att få
möjlighet till nattillsyn via trygghetskamera. Skillnaden mot personlig
nattillsyn är att personalen gör tillsynen via en kamera istället för via ett
hembesök. Denna rutin har tagits fram för att tydliggöra vem som gör vad i
dessa ärenden.

## Roller och ansvar

- Biståndshandläggare tar beslut om insatsen tillsyn samt lämnar
besked om avslut
- EC / Koordinator tar emot beställning och verkställer samt avslutar
verkställighet.
- EC / Koordinator informerar larmgruppen
- Larmgruppen hanterar den praktiska installationen och avslutet
- EC / Koordinator schemalägger e-tillsyn i Phoniro.
- IT-funktion / Vof ansvarar för den tekniska miljön,
systemadministration,  intern support, avtal samt drift- och
leverantörsuppföljning.

## Område
Vad föregår installationen av trygghetskamera, hur gör man samt vad
händer vid avslut.

På Botweb finns:
- informationsbladet ”Trygghetskamera” som riktar sig till brukare.
- Samtyckesblankett som ska skrivas under av brukaren.
- bilaga 1 som tydliggör vilka tider som avses och som är
underskriven av EC.
- bilaga 2 som tydliggör vad nattillsyn är.

2/3
- bilaga 3 som informerar vart brukaren kan vända sig om problem
uppstår.
- Ansvarsförbindelse med tillhörande bild på vad paketet med
kameran innehåller. Denna ska skrivas under av brukaren.

Lathundar för e-tillsyn:
- Administration och uppföljning i Phoniro Care
- Installation hos brukare
- Utförande av e-tillsyn

## Utförande

Beslut, beställning och installation

1. Biståndshandläggare tar beslut om insatsen tillsyn och skickar
beslutet i verksamhetssystemet.
2. EC / Koordinator tar emot beställningen i verksamhetssystemet.
3. EC / Koordinator meddelar larmgruppen via verksamhetssystemet.
4. EC/ Koordinator kontaktar brukaren och kommer överens tider för
tillsyn.
5. Larmgruppen kontaktar brukaren om tid för installation.
6. Om det endast finns denna insats behöver nycklar inhämtas enligt
ordinarie nyckelrutin. Om larm finns behöver inte nycklar inhämtas.
7. Larmgruppen skriver ned antal överenskomna tillsyner och tidsplan i
bilaga 1 (underskriven av EC) i två exemplar. Ett sparas hos
larmgruppen tillsammans med nyckelkvittens.
8. Vid installation av trygghetskamera får brukaren muntlig och
skriftlig information samt skriver under samtyckesblanketten och
ansvarsförbindelsen. Larmgruppen lämnar kvar bilaga 1 som är
underskriven av EC samt bild på vad paketet med kameran
innehåller och bilaga 3 som informerar om vart brukaren kan vända
sig vid funderingar.
9. Larmgruppen informerar EC/Koordinator att det är klart för start av
tillsyn.
10. EC/Koordinator verkställer beställningen innan larmgruppen
installerar kameran.
11. EC / Koordinator schemalägger e-tillsyn i Phoniro.
12. Koordinator uppdaterar genomförandeplanen med tillsynstider (Kan
delegeras till nattpersonal).

Avslut och avinstallation

- Biståndshandläggaren skickar meddelande till EC/Koordinator när
insatsen tillsyn ska avslutas.

3/3

- Om utföraren får informationen först måste de be brukaren
kontakta biståndshandläggaren så att det sker på ett korrekt sätt.
- Om brukaren inte har möjlighet att göra detta informerar
koordinator eller annan personal till myndighet med
informationen i Lifecare.
- Biståndshandläggaren ansvarar för att säkerställa att insatsen
tillsyn ska avslutas och meddelar sedan utföraren i vanlig
ordning.
- EC / Koordinator avslutar e-tillsyn i Phoniro
- EC / Koordinator meddelar larmgruppen via
meddelandehanteringen i verksamhetssystemet att
trygghetskameran ska avslutas hos brukaren.
- Larmgruppen kontaktar brukaren för att boka tid för avhämtning
av kamerapaketet.
•
Larmgruppen hämtar paketet med trygghetskameran hos
brukaren   och meddelar EC/Koordinator att det är gjort.
- EC/Koordinator avslutar ärendet i verksamhetssystemet.
