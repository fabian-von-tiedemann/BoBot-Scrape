# RUTIN
Backup för schema - hemtjänst.
## Inledning
Det finns tillfällen, både korta och långa, då vi av någon anledning inte
kommer åt systemet och kan se våra scheman och planeringen för besöken
till brukarna. För att säkerställa att brukare får sina planerade besök har vi
skapat denna rutin för hur vi arbetar med manuell backup.
## Roller och ansvar
Enhetschefen ansvarar för att rutinen följs.
Koordinator/Planerare ansvarar för att hantera backuphanteringen.
## Område
Utskrift av schema och planering så att fyra veckors backup finns
tillgänglig.
## Utförande
Koordinator/Planerare skriver ut:
1. Ett (1) blad med åtta (8) personalscheman på två sidor.
2. Ett översiktsschema – sätts i pärm som förvaras i dokumentskåp.
Detta löper 4 veckor bakåt i tiden så för varje dag/blad som sätts in
tas det äldsta bort.
3. Brukares beslutade och planerade besökstillfällen – skrivs ut och
uppdateras i brukarens akt.
