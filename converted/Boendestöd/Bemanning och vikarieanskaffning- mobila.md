# RUTIN
Bemanning och vikarieanskaffning inom Mobila teamet

## Inledning
Mobila teamet ska vara bemannad med den personalstyrka som behövs för
att uppdraget ska kunna utföras med god kvalitet och i enlighet med
brukarens önskemål. Denna rutin finns för att tydliggöra hur medarbetare
ska säkerställa att uppdraget utförs även vid personals frånvaro.

## Roller och ansvar
- Enhetschef har det yttersta ansvaret för att det finns ett schema som
säkerställer att brukares behov tillgodoses och att det finns en
kontinuitet i personalflödet.
- Administratörens arbetsuppgift är att sköta schemaplaneringen och
att beställa vikarier vid behov.

## Område
Om inte Administratör hanterar tillsättningen av akut vikarie så gör
medarbetare det.

## Utförande
1. Se över behovet för bemanningen – behöver hela passet
tillsättas?
2. Administratör eller annan behörig beställare skickar en
beställning av vikarie i timecare pool till bemanningsenheten.
3. Låna personal av andra enheter/avdelningar.
4. Vid kris på personal – använd rutin ”prioriteringslista” för vilka
insatser som kan kortas av eller tas bort eller flyttas på.
