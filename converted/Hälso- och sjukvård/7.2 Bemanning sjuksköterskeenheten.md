# RUTIN
7.2 Bemanning sjuksköterskeenheten
## Inledning
Sjuksköterskeenheten bedriver verksamhet dygnet runt och det behöver vara
tydligt vem som ansvarar för bemanningen på dygnets alla timmar.
Prioriteringsordning för arbetsuppgifter vid underbemanning behöver också
tydliggöras.
## Roller och ansvar
Enhetschef: huvudansvar för rutinen och ansvar för att rutinen är känd och
väl implementerad bland medarbetare
Medarbetare: ansvar för att följa rutinen och att uppmärksamma enhetschef
om det finns oklarheter eller svårigheter att följa rutin
## Område
Rutin gäller sjuksköterskeenheten dygnet runt.
## Utförande
Dagtid vardagar
Administratör för sjuksköterskeenheten är bemanningsansvarig vardagar
dagtid och planerar bemanningen för alla sjuksköterskor inklusive all
planerad frånvaro och akut frånvaro som framkommer vardagar mellan kl.
08-16. Administratör ansvarar för bemanningstelefon som är i bruk vardagar
kl. 08-16. Om administratör inte är i tjänst ansvarar rehabenhetens
administratör i första hand och sjuksköterskeenhetens biträdande enhetschef
i andra hand för bemanningen och bemanningstelefonen.

Planering av bemanning ska ske med lång framförhållning. Rutin för
sjukanmälan samt ansökan om ledigheter finns. Bemanningsansvariga
planerar och fördelar medarbetare:
- På de enheter där det är tre ordinarie sjuksköterskor får de vid luckor
gå på två sjuksköterskor.
- På enheter där de är två och någon utav sjuksköterskorna har
planerade ledigheter enstaka dagar i veckan, planeras arbetet och
luckor täcks i mån av tillgängliga medarbetare.
- Vid akut sjukdom kan ordinarie medarbetare behöva täcka upp på en
annan enhet.

- Vid luckor som behöver täckas kontaktas först timanställda
medarbetare, sedan bemanningsföretag och eventuella ordinarie
medarbetare som kan arbeta extra/övertid.
Jourtid
Vid frånvaro/lucka till exempel på grund av akut sjukdom under jourtid är
det sjuksköterska jourtid som bemannar enligt nedan:
1. Ring sjuksköterka från vikarielistan (se lista på mobila jourteamets
kontor).
2. Maila till samtliga bemanningsföretag (se lista på mobila
jourteamets kontor). Enligt avtal måste vi kontakta dem alla
samtidigt.
3. Ring ordinarie kollega (se lista på mobila jourteamets kontor).
4. Ring trygghetsjouren, se telefonnummer på listan över
bemanningsföretag.
Om sjuksköterska från bemanningsföretag inte dyker upp som planerat, ring
bemanningsföretaget.
Om våra egna timvikarier inte dyker upp, ring till dem.
Vid ansträngt läge
Vid hög arbetsbelastning eller hög frånvaro som ger ett ansträngt läge på
specifika enheter, ser bemanningsansvarig tillsammans med
enhetschef/biträdande enhetschef över bemanningen i hela verksamheten.
Ordinarie sjuksköterskor kan behöva flyttas för att täcka upp i ansträngda
verksamheter och ersättas av timanställda eller bemanningssjuksköterskor
på de lugnare enheterna. Vid långvarigt ansträngt läge kan beordring bli
aktuellt för att jämna ut arbetsbelastning.
Prioriteringsordning
Vid hög arbetsbelastning behöver arbetet prioriteras enligt nedan:
1. Akuta insatser
2. Vård i livets slut
3. Säker läkemedelshantering
Samt dokumentationen av ovanstående. Planerade möten och insatser som
kan anstå får skjutas upp eller planeras om till annan dag.
