# RUTIN
Tillvägagångssätt vid inbrott vård- och
omsorgsboende - rutin
## Inledning
Syftet med denna rutin är att tydliggöra vad och hur vi ska agera vid stöld
och inbrott i lokaler som tillhör ett vård- och omsorgsboende. Denna rutin
följer instruktioner tillgängliga via länken: Vid kris, olycka, brand eller brott
- Startsidan – Botwebb (botkyrka.se)
## Roller och ansvar
Rutinen gäller för samtliga vård- och omsorgsboenden i Botkyrka kommun.
Verksamhetschef – huvudansvar för rutinen.

Enhetschef – ansvar för att rutin är känd bland medarbetare och att rutin är
väl implementerad i verksamheten.

Medarbetare - ansvar för att följa rutin och att uppmärksamma enhetschef
om det finns oklarheter eller svårigheter att följa rutin.
## Område
Stöld och polisanmälan.
## Utförande
Den första medarbetare som upptäcker brott ska agera skyndsamt och
kontakta polis. Vid pågående inbrott ska polisen kontaktas på 112 och vid
konstaterat inbrott eller misstanke om genomfört inbrott ska polisen
kontaktas på 114 14 eller via www.polisen.se
Vi misstanke om inbrott eller konstaterat genomfört
inbrott:
•
Kontakta polis (112, 114 14 eller via polisen.se)
- Tala om vad som hänt, var du arbetar, adress och telefonnummer.
- Skriv ned och spara numret på polisanmälan. Det behövs för att
senare lättare kunna följa upp hur det går med utredningen.
- Följ instruktionerna från polisen. Städa inte upp utan undvik att röra
vid föremål på inbrottsplatsen. Vid en teknisk undersökning kan det
finnas möjlighet att samla in spår från gärningspersonen/personerna.

- Om teknisk utrustning har stulits (mobiler, datorer, Ipads, larm) ska
du omgående kontakta IT-supporten tfn 08-530 610 10 så att
supporten kan spärra/avaktivera SIM-kort e dyl.
- Anmäl händelsen till IT-funktionen VOF så att telefoner inaktiveras
för ev digitala låsvred. Anmälan görs via denna länk: Klicka här för
att komma till formuläret.
- Kontakta enhetschef för den enhet där det varit inbrott och informera
om vad som hänt, vilka åtgärder du vidtagit och nummer på
polisanmälan.
Förhindra inbrott och stöld
För att förhindra inbrott och stöld är det viktigt att alla medarbetare är
medvetna om hur vi kan arbeta brottsförebyggande.
•
När du kommer till din arbetsplats/lämnar din arbetsplats är det
viktigt att alltid försäkra sig om att ytterdörren till enheten går i lås.
•
Under arbetsdagen, se till att dörrar in till/ut från avdelningar, förråd,
tvättstugor eller dyl alltid går i lås när du går in/lämnar sådana
utrymmen.
•
Släpp inte in obehöriga i lokalerna.
- Håll koll på och var varsam om värdesaker, passerkort,
arbetstelefoner och arbetsutrustning.
