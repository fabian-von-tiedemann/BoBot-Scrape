Vård- och omsorgsförvaltningen
Personal utebliven vid arbetspass

## Inledning
Syftet med rutinen är att tydliggöra tillvägagångssättet när personal uteblir från
sitt arbetspass.

## Roller och ansvar
-
Enhetschef ansvarar för bemanning och att göra rutinen känd.
-
Medarbetare ansvarar för att följa momenten i rutinen samt hjälpa och
stötta varandra i tillämpningen av den samma.

## Område
Agerande vid utebliven personal

## Utförande
Om personal inte dyker upp till sitt pass under kontorstid
1. Ring till bemanningen och kontrollera att de har bemannat passet.
Skriv i TEAMS och kolla om du kan få hjälp att lösa situationen,
om inte gå till steg 2.
2. Ring till de övriga verksamheterna för att kolla om någon kan
komma och arbeta passet som är tillfälligt övertalig. Om inte gå till
steg 3.
3. Samarbeta med bemanningen och ring runt till vikarier för att se om
någon kan komma in med kort varsel. Om detta inte går gå till steg
4.
4. Ring till arbetsledare och berätta om situationen denne ser till att
någon blir inbeordrad

Om personal inte dyker upp på sitt pass utanför kontorstid.
1. Ring till de andra enheterna för att säkerställa om det kan finnas nå-
gon som kan gå in och hjälpa till. Skriv i TEAMS och kolla om du
kan få hjälp att lösa situationen. Om vi kan flytta brukaren till en
annan enhet över tiden eller om det kan finnas en annan lösning till
det hela. Om inget finns gå till steg 2.
2. Ring de vikarier som vi har och se om någon kan komma in och ar-
beta med kort varsel. Om ingen kan gå till steg 3
3. Ring till trygghetsjouren och berätta vad ni har gjort och att ni vill
beordra in någon. Samråd med dem om hur ni går vidare med situ-
ationen. Om du blir beordrad kan du inte säga nej.
