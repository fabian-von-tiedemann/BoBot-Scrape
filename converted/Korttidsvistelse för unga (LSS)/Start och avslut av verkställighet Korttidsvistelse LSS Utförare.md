# RUTIN
Start och avslut av verkställighet i ordinärt boende-
Korttidsvistelse enligt LSS Utförare

## Inledning
Syftet med denna rutin är att klargöra ansvar och beskriva arbetsgången för
start och avslut av verkställighet för Korttidsvistelse enligt LSS. Hur
kommunikationen förs mellan myndighet och utförare kommer också att
förtydligas.

## Definitioner
- Utredning: Utredningen är underlag för handläggarens beslut.
- Beslut: Beslutet innehåller besked till brukaren om vilka insatser
hen fått bifall respektive avslag på samt motivering till varför.
Beslutet skickas till brukaren
- Beställning: Beställningen är uppdraget tillsammans med antal
dygn som utföraren får till sitt förfogande för att utföra uppdraget. I
uppdraget får utföraren information om brukarens behov.
Beställningen skickas till utföraren.
- Verkställighet: Alla beställningar ska verkställas av utföraren och
är en bekräftelse till Myndighet på att uppdraget utförs
- Uppföljning: Uppföljning görs för att säkerställa att brukaren får
sina behov tillgodosedda

Avgränsning
Korttidsvistelse enligt LSS, kommunens egen regi

## Roller och ansvar
LSS-handläggare
- Utreder personkretstillhörighet, behov av stöd och fattar beslut om
insatsen
- Skickar beställning i verksamhetssystemet
- Ansvarar för att beställningen med insatserna verkställs
- Meddelar utförare om brukare avlidit
- Skickar ny beställning/avslutar beställning
- Avslutar ärendet

Utförare (enhetschef)
- Tar emot beställning av nytt uppdrag i verksamhetssystemet.
- Verkställer nytt uppdrag i verksamhetssystemet.
- Administrerar start av verkställighet (se processkarta).

2/3
- Meddelar myndighet förändrade behov om brukare inte själv kan ta
kontakten. Kan ske muntligt.
- Meddelar myndighet om brukare avlidit.
- Ändrar eller avslutar verkställighet i enlighet med myndighets
beställning

## Utförande
Start av verkställighet
- Beställning på ny brukare skickas i verksamhetssystemet.
- Utföraren (Enhetschef) verkställer beställning.
- Vid frågor om beställningen kontaktas LSS-handläggaren. Läs
uppdraget innan kontakt sker med handläggare.
- Om utföraren inte kan verkställa beställningen meddelas LSS-
handläggaren skriftligt via meddelandefunktionen i
verksamhetssystemet. Ange orsak. Handläggare skickar en
bekräftelse att den mottagit meddelandet. Om bekräftelse inte har
inkommit skicka mail till handläggaren. Det är viktigt att utföraren
journalför orsak och varje kontaktförsök med brukare samt att
handläggaren har informerats.

Pågående verkställighet
- Utförare (Enhetschef) verkställer ny beställning på aktuell brukare
(avsluta tidigare beställning).
- Om nya beställningen inte kan verkställas meddelas LSS-
handläggaren skriftligt via meddelandefunktionen i
verksamhetssystemet, se punkt fyra ovan i start av verkställighet.
- Om brukare upplever förändrade behov innan ordinarie uppföljning
uppmanas denne att själv ta kontakt med sin handläggare. Om det
inte är möjligt kan stödperson ta kontakt med handläggaren.
Handläggaren tar därefter kontakt med brukaren.

Avsluta verkställighet
- LSS-handläggare avslutar beställningen och skickar meddelande i
verksamhetssystemet.
- Utförare (Enhetschef) avslutar verkställighet utifrån myndighets
beställning.
- Om information att en brukare avlidit når utföraren meddelas LSS-
handläggaren.

3/3
Kommunikation
För att kommunikationen ska vara så effektiv som möjligt är det följande
som gäller:
- Vi använder meddelandehanteringen i verksamhetssystemet.
- Vi ringer i akuta ärenden men följer alltid upp med meddelande.
