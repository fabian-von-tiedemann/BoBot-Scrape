# RUTIN
Prioriteringslista vid personalbrist inom
gruppbostad
## Inledning
Det kan ibland förekomma sådan personalbrist så att alla arbetsuppgifter
inte kan utföras som planerat. Vid sådana tillfällen är det viktigt att
medarbetare inom gruppbostad känner till hur de ska prioritera och att det
finns en prioriteringslista framtagen.
## Roller och ansvar
Medarbetare inom LSS- gruppbostad
- ansvarar för att följa rutinen kring prioritering vid personalbrist

Enhetschef
- ansvarar för att göra prioriteringslistan känd hos med medarbetarna.
- ansvarar för förebyggande samt övergripande arbete.
## Område
Prioritering av arbetsuppgifter inom gruppbostad i samband med
personalbrist.
## Utförande
Prioritering
Vid akut personalbrist är det medarbetarna som gör prioriteringar utifrån
listan. Chef eller BE-jour kontaktas om ytterligare stöd behövs.

Om det inte går att få in en timavlönad via bemanningsenheten behöver
arbetsuppgifterna ses över:
- Mat- och omvårdnadsinsatser ska prioriteras.
- Serviceinsatser bokas om vid behov.
- Meddela den boende att insatsen flyttas eller ändras på.

Förebyggande
✓ Säkerställa att medarbetarna känner till rutin för när enhetschef
respektive BE-jouren ska kontaktas.
✓ Säkerställa att enheten bokar timavlönade via bemanningsenheten
vid personalfrånvaro
✓ Stärka krisberedskapen genom att arbeta förebyggande med
egenkontroller

2/2
✓ Samverka med anhöriga och frivilligorganisationer för avlastning
förebyggande.

Akut

1. Mat-och omvårdnadsinsatser ska prioriteras.
2. Prioritera vilka insatser som ska utföras och vilka insatser som
kan planeras in till en annan dag.
3. Minska tiden hos boende genom att exempelvis slå ihop insatser.
4. Säkerställa och kontrollera att alla boende fått nödvändiga
insatser efter dagens slut.
5. Dela på timavlönade mellan enheterna.
6. Ge nya timavlönade nödvändig introduktion för att kunna utföra
insatserna.
7. Beordra in ordinarie medarbetare, görs av enhetschef eller BE-
jouren
8. Samverka med anhöriga.
9. Dokumentera ej utförda insatser.

Övergripande
Om personalbristen består över en längre tid.

✓ Slå ihop flera insatser i samma besök.
✓ Förkorta introduktionen för nya medarbetare.
✓ Utför de allra nödvändigaste insatserna.
✓ Samverka med övriga enheter och hjälps åt över gränserna.
✓ Samverka med anhöriga, frivilligorganisationer och övriga
kontakter som finns.
✓ Se över hur vi kan få in fler medarbetare. Eventuellt fördela
personalresurser inom hela Vof.
