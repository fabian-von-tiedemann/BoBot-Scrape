2024-08-29

Checklista vid verksamhetsöverlämning

Denna checklista är ett stöd för när enhetschef ska lämna över sin verksamhet till ny enhetschef.
Övrig information vid nyanställning överlämnas av ansvarig verksamhetschef, stödfunktioner och
systemägare gällande våra olika system

Administration

Överlämnande och ny enhetschef - gemensamt ansvar
- Informera långtidsfrånvarande medarbetare att man som chef slutar och vem som kom-
mer ny
✓ Överlämnande chef innan avslut
✓ Ny chef kontaktar medarbetare när den tagit över
- Informera samverkansparter
✓ Till exempel god män/anhöriga, Hälso- och sjukvårdsteamet, daglig verksamhet,
✓ Ny chef lämnar sina kontaktuppgifter till samverkanspartner
- Ändring i Lifecare
✓ Uppdatering av enhetschefens kontaktuppgifter under övrig information i lifecare på
varje boende.

Överlämnande enhetschef
- Ta bort sina kontaktuppgifter från gruppbostadens gemensamma mailgrupp i
lifecare/och gruppbostadens VOF-brevlåda i outlook. För att ta bort VOF brevlåda kon-
takta ansvarig administratör.
- Åtgärda KIA-anmälningar, avvikelser i lifecare, HSL-beställningar.
- Uppdatera Stratsys, fyll i aktiviteter genomförda i SAM-modulen.
- Privata medel
✓ Gör en avstämning på varje boendes privata medel och skriv att du gjort en avstäm-
ning och din signatur på redovisningslista för privata medel
- Rensa och städa kontorsplatsen

Ny enhetschef
- Mailgrupp/meddelande i viva
✓ Lägg till din mail i gruppbostadens gemensamma meddelandegrupp i lifecare.
✓ Beställ behörighet till VOF-brevlåda för outlook genom kontakt med ansvarig admi-
nistratör
- Sammarbetsrum
✓  Behörighet till verksamhetens sammarbetsrum kontakta administratör som lägger till
behörigheten.
✓ Lägg in profilbild i outlook och botwebb

2
Överlämnande chefs information till ny enhetschef om verksamheten
Tänk på att inte skriva sekretesskyddade uppgifter eller känsliga uppgifter på ett sådant sätt att
obehöriga kan riskera att ta del av dem. Vid rehabiliteringsärenden, handlingsplaner och liknande
hänvisa till de dokument som finns i ärendet, till exempel från lönesystem, rehabiliteringssystem
eller fysisk personakt.

Boende:
✓ handlingsplaner
✓ vårdplaner utöver det vanliga
✓ delegeringar som behövs
✓ särskilda överenskommelser
✓ riskbedömningar utifrån den enskilde-riskbedömning av skyddsåtgärder,
✓ brukarundersökning.

Medarbetare:
✓ Sjukskrivningar/rehabiliteringsärenden
✓ övriga personalärenden
✓ handlingsplaner, handlingsplaner utifrån medarbetarundersökning
✓ tjänstledigheter/föräldraledigheter
✓ personallarm utifrån arbetsmiljö
✓ personalakter
✓ senaste russamtalet

Bemanning:
✓ ”tänket” kring schema
✓ vakanta turer
✓ vikarier

Arbetsmiljö generellt:
✓ aktuella arbetsmiljöfrågor,

✓ riskbedömningar.

Verksamhet:
✓ mål/aktiviteter
✓ utbildningsinsatser/behov av utbildningsinsatser
✓ bokade tvärprofessionella teammöten
✓ ansvarsområden
✓ brukarlarm
✓ APT och GPT dag/tid/dagordning visa och lägga in i nya chefens outlook.
✓ eventuella granskningar på gång

Budget:

✓ aktuell budget
✓ ansvar/objekt
✓ behörig beställare
✓ eventuella budgetåtgärder
