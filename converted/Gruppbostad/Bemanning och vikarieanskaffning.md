Bemanning och vikarieanskaffning
## Inledning
Enligt LSS ska gruppbostaden ha tillräcklig bemanning så att uppdraget
utförs med god kvalitet och utifrån brukarnas individuella behov och
önskemål. Denna rutin beskriver hur bemanning säkerställs vid
medarbetares frånvaro.
## Roller och ansvar
Alla medarbetare inom LSS- gruppbostad ansvarar för att följa rutinen.
Medarbetare/administratör
- är delaktig i schemaplanering och att beställa /ringa in vikarier från
bemanningsenheten.

Enhetschef
- ansvarar för att det finns ett schema som säkerställer att brukares
behov tillgodoses och att det finns en kontinuitet i medarbetarflödet.
## Område
Säkerställa att uppdraget utförs med god kvalitet och i enlighet med
brukares individuella behov och önskemål.
## Utförande
För en dialog om vem som tar ansvaret för att tillsätta vikarie, enhetschef,
administratör eller medarbetare.
1. Se över behovet för bemanningen – behöver hela passet
tillsättas?
2. Ring/beställ vikarie via bemanningsenheten
3. Se över möjligheten att låna medarbetare av andra enheter.
4. Vid brist på medarbetare eller vid krissituation – använd rutin
”prioriteringslista vid personalbrist” för vilka insatser som kan
kortas av, tas bort eller flyttas på.
5. Kan varken vikarie eller lånad medarbetare stötta i bemanning,
kontakta enhetschef kl 08:00-16:00, som vid behov kan ta beslut
om att beordra in medarbetare.
6. Utanför ordinarie kontorstid kontaktas BE-jouren på
telefonnummer 08–530 611 60. Vardagar mellan kl. 16:00 och
08:00 samt dygnet runt under helger och röda dagar. BE-jouren
har vid behov möjlighet att fatta beslut om att beordra in
medarbetare.
