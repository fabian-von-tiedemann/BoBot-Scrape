# Requirements: BoBot-Scrape v5.0 QA Generation Pipeline

**Defined:** 2026-01-24
**Core Value:** Generera tusentals validerade QA-par fran kunskapsbasen for AI-assistenttraning

## v5.0 Requirements

Requirements for QA generation pipeline. Each maps to roadmap phases.

### Persona Model

- [x] **PERS-01**: Definiera persona-modell med roll + situation — validated
- [x] **PERS-02**: Skapa 5-10 distinkta personas (underskoterska, hemtjanst, nattjour, ny pa jobbet, etc) — validated
- [x] **PERS-03**: Persona-konfiguration i YAML-fil — validated

### Question Generation

- [x] **QGEN-01**: Generera fragor fran dokumentinnehall med Gemini — validated
- [x] **QGEN-02**: Persona-drivna fragor (formulerade som persona skulle fraga) — validated
- [x] **QGEN-03**: Kalldokumentreferens for varje fraga — validated
- [x] **QGEN-04**: 3-5 fragor per dokument — validated
- [x] **QGEN-05**: Batch-generering med progress-tracking — validated

### Answer Generation

- [x] **AGEN-01**: Svar grundade i kalldokument — validated
- [x] **AGEN-02**: Kallcitat i varje svar (dokumentnamn, sektion) — validated
- [x] **AGEN-03**: Klarsprak (enkel svenska, B1-niva) — validated
- [x] **AGEN-04**: Extraktionsstil (citera relevant text fran kalla) — validated

### Validation

- [x] **VALD-01**: Kallverifiering (svar finns i refererat dokument) — validated
- [x] **VALD-02**: Kvalitetsbedomning (relevans, korrekthet, fullstandighet) — validated
- [x] **VALD-03**: Tvastegs validering med LLM-as-judge — validated
- [x] **VALD-04**: Filtrera bort hallucineringar och lag kvalitet — validated
- [x] **VALD-05**: Validerings-score i output (for granskning) — validated

### Export

- [x] **EXPRT-01**: JSONL-export (HuggingFace-kompatibelt) — validated
- [x] **EXPRT-02**: Metadata per QA-par (persona, kalla, score) — validated
- [x] **EXPRT-03**: Separat fil for godkanda vs underkanda QA-par — validated

### Integration

- [x] **INTG-01**: Standalone CLI (generate_qa.py) — validated
- [x] **INTG-02**: Integration i pipeline.py som valfritt steg — validated
- [x] **INTG-03**: Checkpointing for resume vid avbrott — validated
